# AeroSpace Keybindings

i3-style keybindings via window mode (`=`-hold via kanata).

## Window Mode (`=`-hold)

| Binding        | Action            |
| -------------- | ----------------- |
| `=+hjkl`       | Focus             |
| `=+Shift+hjkl` | Move window       |
| `=+1-9`        | Workspace         |
| `=+Shift+1-9`  | Move to workspace |
| `=+f`          | Fullscreen        |
| `=+q`          | Close             |
| `=+t`          | Terminal          |
| `=+r`          | Resize mode       |
| `=+;`          | Service mode      |

## App Launchers (Hyper = Tab-hold)

Same mnemonics as LeftWM `Super+key`.

| Binding   | App       |
| --------- | --------- |
| `Hyper+t` | Alacritty |
| `Hyper+b` | Firefox   |
| `Hyper+m` | Spotify   |
| `Hyper+o` | Obsidian  |
| `Hyper+s` | Slack     |

## Alt (non-conflicting)

| Binding   | Action     |
| --------- | ---------- |
| `Alt+f`   | Fullscreen |
| `Alt+t`   | Float/tile |
| `Alt+Tab` | Last ws    |
| `Alt+r`   | Resize     |

## Summary

```text
=-hold  = WM control (Ctrl+Cmd, no tmux conflict)
Tab-hold = Hyper (app launchers)
```
