# LeftWM Keybindings

**Mod**: Super (Mod4)

## Navigation

| Binding | Action |
| ------- | ------ |
| `Super+hjkl` | Focus direction |
| `Super+Shift+hjkl` | Move window |
| `Super+1-9` | Workspace |
| `Super+Shift+1-9` | Move to workspace |

## Window Actions

| Binding | Action |
| ------- | ------ |
| `Super+f` | Fullscreen |
| `Super+Shift+q` | Close |
| `Super+Shift+Ret` | Terminal |
| `Super+Space` | Rofi |
| `Super+Tab` | Last workspace |
| `Super+Ctrl+jk` | Cycle layouts |
| `Super+Shift+r` | Reload |
| `Super+Shift+x` | Exit |

## Cross-Platform Note

On macOS, AeroSpace uses `Alt` instead of `Super` for the same bindings.
The Kanata `=` layer sends `Ctrl+Alt`, which tmux/zellij catch (not AeroSpace).
